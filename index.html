Mongoose
ODM

Elegant MongoDB object modeling for Node.js

Flexible, schema based, and feature-rich, Mongoose solves common problems for real-world applications. Its fully customizable middleware, exposed object lifecycle, and rich query syntax make developing applications a breeze.
